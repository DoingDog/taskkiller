# taskkiller
kills one (or more)process that loop starts itself

how to use it?
Place tk.bat into System32,and use command in cmd like

tk explorer.exe
or
tk thevirus*

and the process will be killed
 or simply use tk to kill more processes at once like
 
 tk
 explorer.exe
 virus*
 virus2*
 (enter)
 
 press enter to exit the batch
